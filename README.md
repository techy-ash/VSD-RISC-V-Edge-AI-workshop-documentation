# RISC-V Edge AI Workshop Documentation

## About

This repository documents my journey through the **VSD IAT RISC-V Edge AI** course, where I explored the deployment of Machine Learning models on resource-constrained embedded systems.

Throughout the course, I progressed from understanding the mathematical foundations of Machine Learning to implementing optimized inference pipelines suitable for embedded RISC-V processors. The labs covered optimization techniques, classical machine learning algorithms, quantization, and deployment of trained models in Embedded C

Rather than simply completing assignments, I focused on understanding how machine learning models transition from Python-based experimentation to efficient execution on edge hardware.

-Note: Since I don't have the VSDSquadron Pro board, I have utilised SifiveStudio in the GitHub virtual lab.

# Learning Journey

Each laboratory introduced a new stage of the embedded AI workflow.

The course began with understanding optimization through Gradient Descent, followed by implementing a KNN classifier to build intuition about supervised learning. The focus then shifted towards Support Vector Machines, where I explored model training, quantization, and deployment using integer arithmetic suitable for embedded environments. Finally, I implemented Neural Networks and studied their inference process and architecture.

One of the most valuable takeaways from this course was understanding how machine learning models can be optimized and deployed beyond desktop environments onto embedded processors with limited computational resources.
