# Lab 2 — K-Nearest Neighbors Classifier

## Objective

Implement and evaluate the K-Nearest Neighbors (KNN) algorithm for supervised classification.

---

## Overview

In this lab, I explored one of the simplest machine learning classifiers. Rather than learning model parameters, KNN classifies new samples by comparing them with nearby training samples.

The implementation reinforced concepts such as feature spaces, distance metrics, and decision boundaries.

## What I Learned

- Distance-based classification
- Euclidean distance computation
- Effect of different K values

## Challenges

Selecting an appropriate value of K significantly affected prediction accuracy. Comparing different values helped understand the trade-off between bias and variance.

---

## Key Takeaway

KNN demonstrated how effective simple algorithms can be when supported by appropriate preprocessing and feature representation.
