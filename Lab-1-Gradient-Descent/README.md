# Lab 1 — Gradient Descent

## Objective

The objective of this lab was to understand Gradient Descent as the fundamental optimization algorithm used in Machine Learning.

---

## Overview

Before working with classifiers and neural networks, it is essential to understand how model parameters are optimized. This lab introduced Gradient Descent by minimizing a cost function iteratively through parameter updates.

I implemented the algorithm in Python while observing how different learning rates influence convergence.

---

## Workflow

Dataset

↓

Initialize Parameters

↓

Compute Cost

↓

Calculate Gradients

↓

Update Parameters

↓

Repeat Until Convergence

---

## What I Learned

Through this exercise, I gained practical understanding of:

- Cost functions
- Learning rate selection
- Gradient computation
- Convergence behaviour
- Optimization in Machine Learning

I also observed how improper learning rates can either slow convergence or cause divergence.

---

## Challenges

Initially, understanding how gradients affected parameter updates was conceptually challenging. Visualizing the cost function helped connect the mathematical equations with the optimization process.

---

## Key Takeaway

This lab established the mathematical foundation required for understanding more advanced machine learning algorithms later in the course.
